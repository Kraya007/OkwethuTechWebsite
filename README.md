# OkwethuTech Official Website

The OkwethuTech website is mostly content-driven with only the contact page having a contact form that users can use to communicate with the company. As such, the decision taken for which rendering strategy to use is ISR - Incremental Static Regeneration.

## The website is built using:

- HTML5 for content
- [Astro](https://astro.build/) framework for optimization (SEO, increased loading speed)
- Typescript and Astro Island for User Interactivity
- TailwindCss for styling

## Installation

```bash
# clone to directory
git clone git@github.com:nickypelo/okwethuTech-Website.git

# enter okwethuTech-Website directory
cd okwethuTech-Website

# install dependencies
npm install

# run application
npm run dev

```
